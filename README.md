# GasGuard-AI-IIoT-Predictive-Leak-Defense
IOT system using MQTT sensor fusion. Gas &amp; pressure data feed an ESP, triggering an n8n workflow where a Python AI engine detects anomalies n8n signals a solenoid actuator for automatic valve closure and alert Discord. Seamless  integrates predictive maintenance and industrial automation to ensure pipeline safety and prevent catastrophic failures.
